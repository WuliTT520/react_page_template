# React_Page_Template
这是一个由 `react-react-app` 改造的多入口React应用模板。

### 安装
Clone到本地以后 使用
> npm install 

进行安装
(ps:将package.json 中项目的名称改成自己项目的名称)
### 运行
> npm start

### 打包
> npm run build

### 浏览
> http://localhost:3000/page1.html/ <br>
> http://localhost:3000/page2.html/

### 版本
>create-react-app 4.0.1
